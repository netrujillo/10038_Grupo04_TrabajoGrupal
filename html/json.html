información general

¿Qué es?

JSON (JavaScript Object Notation) es un formato ligero de intercambio de datos que se utiliza para representar estructuras de datos simples y complejas. Es ampliamente utilizado en aplicaciones web y servicios web como un formato común para transmitir datos entre el servidor y el cliente.

Estructura

Constituido por dos estructuras:

Una colección de pares de nombre/valor. En varios lenguajes esto es conocido como un objeto, registro, estructura, diccionario, tabla hash, lista de claves o un arreglo asociativo.

Una lista ordenada de valores. En la mayoría de los lenguajes, esto se implementa como arreglos, vectores, listas o sequencias

Características

1. Sintaxis sencilla JSON utiliza una sintaxis simple basada en pares de clave-valor.
2. Legilibilidad JSON es fácilmente legible tanto para los humanos como para las máquinas.
3. Ligero JSON es un formato de datos ligero, lo que significa que su tamaño de archivo es relativamente pequeño en comparación con otros formatos como XML.
