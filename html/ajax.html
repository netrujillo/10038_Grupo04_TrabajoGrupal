información general

¿Qué es?

AJAX (Asynchronous JavaScript And XML) es una técnica de programación que permite actualizar partes específicas de una página web sin necesidad de recargarla por completo. AJAX combina varias tecnologías, como JavaScript, XML (aunque también puede utilizar otros formatos de datos, como JSON) y el objeto XMLHttpRequest, para lograr la comunicación asíncrona con el servidor y actualizar el contenido dinámicamente.

Composición

  • XHTML y CSS para presentar información
  • DOM para visualizar e interactuar de forma dinámica la información presentada
  • XMLHttpRequest para manipular los datos de forma asíncrona con el servidor web
  • XML, HTML y XSLT para el intercambio y la manipulación de datos.

Características

1. Actualización dinámica Permite cargar y mostrar datos actualizados en una página web sin tener que recargarla completamente.
2. Interacción Asincrónica La comunicación con el servidor se realiza en segundo plano sin interrumpir la interacción del usuario con la página.
3. UX Proporciona una respuesta más rápida al usuario, ya que solo se actualiza la parte necesaria de la página en lugar de recargarla por completo.
