Semejanzas y Diferencias

XML-JSON

SEMEJANZAS DIFERENCIAS
Los datos en JSON como en XML se pueden obtener mediante el uso de un XMLHttpRequest. JSON es fácil de leer ya que se basa en definir objetos y sus valores, XML no tanto porque posee etiquetas de marcado.
Los formatos de ambos son de tipo texto plano. XML requiere de una mayo cantidad de código en comparación a JSON.
Ambos soportan unicode, por tanto la son compatibles con intercionalización. JSON es más fácil de crear así como la sixtaxis de la codificación lo es, XML a diferencia tiene un poco más de sintaxis por aprender.
Tanto JSON como XML son jerárquicos, por tanto existen valores dentro de otros en una jerarquía específica. XML es similar a programar por tanto requiere más conocimiento, JSON trabaja con un limitado rango de tipos de dato que puede no ser suficiente para todas las aplicaciones.
JSON y XML son autodescriptivos, dado que tanto los datos de ambos son texto legible por humano. JSON es u subconjunto de JS por tal razón puede ser usado para correr código malicioso, XML en cambio es más seguro.
